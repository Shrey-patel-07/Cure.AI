How to use it?
1. Clone this repo.
2. cd Backend
3. npm install
4. npm start
5. cd ../frontend
6. npm install
7. npm run dev
8. cd ../Flask-Backend
9. python -m venv venv
10. source venv/bin/activate
11. pip install -r requirements.txt
12. (models are stored in git lfs. may need to download manually).
13. python app.py
14. open http://localhost:5713
